# API Testing Playground

A free, open-source web tool for testing REST APIs instantly. No signup required, no API keys needed.

## Features

- 🌐 **Test any REST API**: GET, POST, PUT, DELETE, PATCH
- 📦 **Full request control**: Headers, parameters, body, authentication
- ⚡ **Real-time results**: Status codes, response time, size
- 🎨 **Beautiful interface**: Clean, modern, responsive design
- 🔒 **Privacy first**: All requests run in your browser, no server logging
- 🆓 **Completely free**: No limits, no paywalls

## Usage

1. Visit the live site: [https://yourusername.github.io/api-testing-playground](https://yourusername.github.io/api-testing-playground)
2. Enter your API endpoint
3. Set method, headers, parameters, body as needed
4. Click "Send Request"
5. View response with status, timing, and formatted output

## Live Demo

Try it with the built-in example:
- URL: `https://jsonplaceholder.typicode.com/posts`
- Method: `POST`
- Body: Pre-filled example JSON

## Technology

- Pure HTML, CSS, JavaScript
- No frameworks, no build tools
- Deploys anywhere (GitHub Pages, Netlify, Vercel, etc.)
- Works offline (after first load)

## Privacy & Security

- **No backend server**: All requests run in your browser
- **No logging**: We don't store any request data
- **No tracking**: No analytics, no cookies
- **CORS aware**: Works with APIs that allow browser requests
